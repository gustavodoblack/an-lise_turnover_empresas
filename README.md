# an-lise_turnover_empresas
Projeto de análise exploratória de turnover utilizando Python, Pandas e Seaborn. Foram realizadas etapas de ETL, validação de dados, estatística descritiva e visualizações para investigar possíveis fatores associados à evasão de funcionários. O estudo demonstra raciocínio analítico estruturado e interpretação estatística orientada a dados.
